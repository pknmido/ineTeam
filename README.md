# ineteam

A modern, cross-platform mobile application for matchmaking. Built using the Flutter SDK and Firebase.


## Getting Started

Before you begin, ensure you have the following installed:
* [Flutter SDK](https://flutter.dev) (latest stable version)
* [Dart SDK](https://dart.dev)
* Node.js (18.x or 20.x)
* Firebase CLI
```bash
npm install -g firebase-tools
```
* A code editor like [VS Code](https://visualstudio.com) or [Android Studio](https://android.com)

## 1.Clone the repository
```bash
git clone https://github.com/Ghass-1/ineTeam
cd ineTeam
```
## 2.Install dependencies
```bash
flutter pub get
```
## 3.Configure the Cloud Functions Backend
```bash
cd functions
npm install
cd ..
```
## 4.Run the project
Use an emulator or a browser to your choosing (chrome preferred):
```bash
flutter run -d chrome
```
